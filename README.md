# PaperPlot AI

这是一个专业的图表绘制工具，适用于演示文稿和学术论文，通过 Gemini AI 从文本描述生成图表。

在 AI Studio 中查看应用: https://ai.studio/apps/temp/1

## 本地运行

**前置要求:** Node.js

1. 安装依赖:
   ```bash
   npm install
   ```
2. 在 [.env.local](.env.local) 文件中设置 `GEMINI_API_KEY` 为你的 Gemini API 密钥
3. 运行应用:
   ```bash
   npm run dev
   ```
